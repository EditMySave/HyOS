## Prerequisites

- **Docker** and **Docker Compose** v2+
- **Hytale account** with dedicated server access
- **8 GB RAM** minimum (4 GB for the server, headroom for the OS)
- **x86_64** architecture (ARM is not supported by Hytale)

## Container Images

| Image | Description |
|-------|-------------|
| `ghcr.io/editmysave/hyos/server:latest` | Hytale dedicated server |
| `ghcr.io/editmysave/hyos/manager:latest` | Web management dashboard |
